# WebDevBootCamp

> YelpCamp project based on Colt Steele

# Set up
```
npm i all
```
# Server
open window powershell
```
type mongod
```
open another window powershell
```
type mongo
```
# Run the app by open Git Bash
```
node app.js
visit the site: localhost:3000
```
